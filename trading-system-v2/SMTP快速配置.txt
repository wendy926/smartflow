📧 Gmail SMTP 快速配置指南
============================================

【第一步】获取 Gmail 应用专用密码

1. 访问：https://myaccount.google.com
2. 登录您的 Gmail（wendy.wang926@gmail.com）
3. 左侧菜单 → 安全性
4. 找到"两步验证" → 如果没有启用，先启用它
5. 在"两步验证"部分 → 点击"应用专用密码"
6. 选择"邮件"和"其他设备"
7. 输入自定义名称（如：SmartFlow）
8. 点击"生成"
9. **复制生成的 16 位密码**（格式：xxxx xxxx xxxx xxxx，但使用时去除空格）

【第二步】配置 VPS

SSH 连接到 VPS：
ssh -i ~/.ssh/smartflow_vps_new root@47.237.163.85

执行以下命令：
cd /home/admin/trading-system-v2/trading-system-v2/trading-system-v2

# 备份配置
cp .env .env.backup.$(date +%Y%m%d_%H%M%S)

# 添加 SMTP 配置（将 YOUR_16_CHAR_PASSWORD 替换为实际密码，去除空格）
cat >> .env << 'EOFCONFIG'

# Gmail SMTP 配置
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=wendy.wang926@gmail.com
SMTP_PASS=YOUR_16_CHAR_PASSWORD_HERE
EOFCONFIG

# 重启服务
pm2 restart main-app

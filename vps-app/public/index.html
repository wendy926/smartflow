📋 交易策略规则说明

🎯 趋势判断 (日线级别)

多头趋势条件：

  • MA20 > MA50 > MA200 (移动平均线多头排列)
  • 当前价格 > MA20 (价格在短期均线之上)
  • 满足以上条件时显示 "多头"

空头趋势条件：

  • MA20 < MA50 < MA200 (移动平均线空头排列)
  • 当前价格 < MA20 (价格在短期均线之下)
  • 满足以上条件时显示 "空头"

震荡趋势：

  • 不满足多头或空头条件时显示 "震荡"

📊 信号判断 (小时级别确认)

多头信号条件：

  • 日线趋势为 "多头" 或 "震荡"
  • 成交量倍数 > 1.5 (当前成交量/平均成交量)
  • 资金费率绝对值 ≤ 0.001 (避免高费率)
  • VWAP确认：价格 > VWAP
  • 20周期高低点突破：当前价格 > 过去20小时最高价
  • 持仓量6小时变化 > 0 (资金流入)
  • CVD数据可用且方向为多头

空头信号条件：

  • 日线趋势为 "空头" 或 "震荡"
  • 成交量倍数 > 1.5
  • 资金费率绝对值 ≤ 0.001
  • VWAP确认：价格 < VWAP
  • 20周期高低点突破：当前价格 < 过去20小时最低价
  • 持仓量6小时变化 < 0 (资金流出)
  • CVD数据可用且方向为空头

无信号：

  • 不满足多头或空头信号条件时显示 "无信号"

⚡ 入场执行 (15分钟级别)

多头入场条件：

  • 小时信号为 "多头"
  • 价格回调至EMA20或EMA50附近
  • 识别setupHigh：前一根15分钟K线的最高价
  • 当前价格突破setupHigh
  • 满足以上条件时显示 "多头入场"

空头入场条件：

  • 小时信号为 "空头"
  • 价格回调至EMA20或EMA50附近
  • 识别setupLow：前一根15分钟K线的最低价
  • 当前价格跌破setupLow
  • 满足以上条件时显示 "空头入场"

等待入场：

  • 不满足入场条件时显示 "等待入场"

💰 风险控制指标

现价止损距离：

  • 计算公式：(当前价格 - 止损价格) / 当前价格
  • 用于衡量止损位置与当前价格的距离

最大杠杆倍数：

  • 计算公式：1 / (现价止损距离 + 0.5%) 向下取整
  • 基于止损距离计算的最大安全杠杆

最低保证金：

  • 计算公式：200 / (最大杠杆倍数 × 止损距离) 向上取整
  • 基于杠杆和止损距离计算的最低保证金要求

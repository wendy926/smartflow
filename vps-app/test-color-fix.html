🎯 技术实现

按钮修复：
• 从 rgba(255, 255, 255, 0.2) 改为 linear-gradient(45deg, #667eea, #764ba2)
• 添加 box-shadow 提升视觉层次
• 保持白色文字，确保对比度
输入框修复：
• 添加 box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1)
• 增强边框 border: 2px solid #e0e0e0
• 确保文字颜色 color: #333
容器修复：
• 从 rgba(255, 255, 255, 0.95) 改为 white
• 添加边框 border: 1px solid rgba(102, 126, 234, 0.1)
• 移除 backdrop-filter 避免模糊
